# Dotfiles

</div>

### Install.

```bash
BINDIR=~/.local/bin sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply https://github.com/nickcorin/dotfiles.git
```

### Manually applying changes.

```bash
# Retry package installation.
chezmoi apply

# Force reinstall all packages.
chezmoi apply --force
```
